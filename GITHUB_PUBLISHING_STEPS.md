# Step-by-Step Guide: Publishing Your RTP Website on GitHub Pages (FREE)

Follow these simple steps to publish your website for free on GitHub Pages.

## Prerequisites
- A GitHub account (free at https://github.com)
- Your website files ready in the `rtp-community-website` folder

---

## Step 1: Create a GitHub Repository

1. **Go to GitHub.com** and sign in (or create a free account)

2. **Click the "+" icon** in the top right corner
   - Select **"New repository"**

3. **Fill in the repository details:**
   - **Repository name:** `rtp-community-website` (or any name you like)
   - **Description:** "TSA Website Design Competition - RTP Community Story"
   - **Visibility:** Select **"Public"** (required for free GitHub Pages)
   - **DO NOT** check "Initialize with README" (we'll upload files)
   - Click **"Create repository"**

---

## Step 2: Upload Your Website Files

### Option A: Using GitHub Web Interface (Easiest)

1. **On your new repository page**, you'll see a page that says "Quick setup"

2. **Click "uploading an existing file"** link

3. **Drag and drop ALL files** from your `rtp-community-website` folder:
   - `index.html`
   - `history.html`
   - `lately.html`
   - `culture.html`
   - `economy.html`
   - `future.html`
   - `styles.css`
   - `script.js`
   - `map.js` (if you have it)
   - Any other files

4. **Scroll down** and click the green **"Commit changes"** button
   - Leave the commit message as is or write: "Initial website upload"

### Option B: Using GitHub Desktop (Recommended for updates)

1. **Download GitHub Desktop** from https://desktop.github.com (free)

2. **Sign in** with your GitHub account

3. **Click "File" ‚Üí "Add Local Repository"**

4. **Click "Choose"** and select your `rtp-community-website` folder

5. **Click "Publish repository"**
   - Make sure "Keep this code private" is **UNCHECKED**
   - Click "Publish repository"

---

## Step 3: Enable GitHub Pages

1. **Go to your repository** on GitHub.com

2. **Click "Settings"** tab (at the top of the repository)

3. **Scroll down** to find **"Pages"** section (in the left sidebar)

4. **Under "Source"**, click the dropdown and select **"main"** branch

5. **Click "Save"**

6. **Wait 1-2 minutes** for GitHub to build your site

7. **Refresh the page** - you'll see a green box with your website URL:
   ```
   https://yourusername.github.io/rtp-community-website/
   ```

---

## Troubleshooting

### Website not showing up?
- **Wait 2-3 minutes** - GitHub needs time to build
- **Check the URL** - Make sure you're using the correct one from Settings ‚Üí Pages
- **Check repository is Public** - GitHub Pages only works with public repos (free tier)

### Images not loading?
- Make sure all image files are uploaded
- Check file paths in HTML are correct
- Use relative paths like `images/photo.jpg` not absolute paths

### Map not working?
- Google Maps embed should work automatically
- If issues, check that iframe code is correct in HTML files

### Changes not appearing?
- **Clear your browser cache** (Ctrl+F5 or Cmd+Shift+R)
- Wait 2-3 minutes after pushing changes
- Check that files were actually committed
